Technology Stack

Backend: Java 17, Jakarta Servlet 6.0, Jakarta JSP

Database: MySQL 8.0+

Build Tool: Maven 3.6+

Frontend: JSP, JSTL, CSS3 with Gaming Theme

Quick Start

1. Set up Database: Run database.sql in MySQL

2. Configure Connection: Update app.properties with your database details

3. Build Project: Run mvn clean package

4. Deploy to Tomcat: Copy WAR to TOMCAT_HOME/webapps/

5. Access Application: Visit http://localhost:8080/eSportsTeamManagementSystem/
